🔥 Firebase Realtime Database

เชื่อมต่อ Firebase เพื่อให้ข้อมูลทุกคนซิงค์กัน Admin สามารถเห็นผลการสอบของนักศึกษาทุกคน

วาง URL ที่ได้จาก Firebase Console → Realtime Database
🟢 เชื่อมต่อสำเร็จ! Firebase พร้อมใช้งาน
วิธีสร้าง Firebase Realtime Database:
1. ไปที่ console.firebase.google.com
2. สร้างโปรเจกต์ใหม่ → Realtime Database → สร้าง → เลือก Test mode
3. คัดลอก URL (เช่น https://my-app-default-rtdb.firebaseio.com) ใส่ในช่องด้านบน
4. กด 💾 บันทึก URL → 🔌 ทดสอบ → รีเฟรชหน้า
